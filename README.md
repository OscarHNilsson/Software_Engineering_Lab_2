Overview

A Jetpack Compose-based Android app for controlling smart home devices (light, door, window) via voice commands and manual toggles. The states are stored in a Firebase RealTime Database

Prerequisites
- Android Studio (latest version).
- Firebase project with Realtime Database enabled.

Steps
- Setup a simulated android device through Android Studio.
- Open the project, sync gradle and build.
- Launch app

Usage
- Manual control through switches for the different devices.
- Voice activation through button press, use commands like "Turn on the light" or "Close the door" to interact with the system.
